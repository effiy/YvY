## 调色板 Color Palette[](https://comfyui-wiki.com/zh/interface/appearance#调色板-color-palette)
![ComfyUI 颜色主题](https://comfyui-wiki.com/_next/static/media/color-palette.c718df46.jpg) 调色板主要用于ComfyUI 整体的颜色主题配置，对应菜单功能为
  1. 切换 ComfyUI 主题
  2. 将当前选中的主题导出为`Json` 格式
  3. 从`Json`文件中载入自定义主题配置
  4. 删除自定义主题配置


目前预设的主题主要有：
  * Dark 暗色主题
  * Light 明亮主题
  * Arc主题
  * Solarized 过度曝光主题
  * Nord 主题
  * Github 风格主题
