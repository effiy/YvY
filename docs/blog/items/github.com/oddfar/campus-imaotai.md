## 项目介绍
[](https://github.com/oddfar/campus-imaotai#项目介绍)
i茅台app，每日自动预约茅台
  * 平台注册账号
  * 添加多个用户
  * 自动预约
  * 类型选择（本市出货量最大的门店，或位置附近门店）
  * 自动旅行
  * 首次旅行分享
  * 获取申购耐力值
  * 自定义时间/随机时间预约或旅行
  * 申购结果消息推送


此项目使用 **Campus** 进行编写：<https://github.com/oddfar/campus>
